# مشروع إبراهيم الحربي

واجهة عربية شخصية مع بوابة دخول مدمجة للمؤثر ولوحة تحكم داخل نفس المسار.

## الملفات الحالية

- `index.html`: الواجهة العامة وبوابة دخول الإدارة.
- `admin/index.html`: لوحة التحكم الحالية.

## التنظيم الحالي

- هوية بصرية موحدة: أبيض، أسود، رمادي.
- مسار عربي فقط في الواجهة العامة.
- دخول الإدارة من نفس الرابط عبر نافذة مدمجة.

## خطوات التطوير المقترحة

1. فصل CSS و JavaScript إلى ملفات مستقلة.
2. نقل منطق البيانات المشترك إلى وحدات منظمة.
3. تحسين الأمان وتغيير آلية كلمة المرور.
4. إضافة أدوات تنسيق وفحص كود.

## ملاحظات

- لتشغيل المشروع على ويندوز بدون فتح ملف `npm.ps1` في المفكرة، شغّل الملف `start-project.cmd` مباشرة. هذا الملف يستخدم `node dev-server.js` ولا يعتمد على `npm`.
- ملف `.git/` هو مجلد نظامي خاص بـ Git ولا يُنشأ يدويًا كملف نصي.
- ملف البيئة المحلي المقترح للمستقبل هو `.env.local` ولا ينبغي رفعه للمستودع.
